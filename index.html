This is a web tool that helps the user to find the solution to the Knapsack Problem, i.e., to find the resultant profit from a given set of items each with a weight and value.


The fractional Knapsack Algorithm determines the total number of items that can be included in a collection or a knapsack, so that the total weight is less than or equal to a given limit and the total value is as large as possible. In 0-1 Knapsack Algorithm however, the items cannot be divided, which means that it should either take the item as a whole or leave it completely
